# 🚀 PrepMe – AI-Powered Interview Preparation Platform

PrepMe is a modern interview preparation platform designed to help students and job seekers practice interviews, improve communication skills, and boost confidence using AI-driven mock interview sessions.

## ✨ Features

* 🎯 AI-powered mock interviews
* 📝 Real-time interview questions based on selected roles
* 🤖 AI-generated feedback and performance analysis
* 📊 Interview history tracking
* 🔐 Secure authentication and user management
* 📱 Responsive and user-friendly interface
* 🌐 Full-stack web application

## 🛠️ Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL

### Authentication

* JWT (JSON Web Tokens)

### AI Integration

* Gemini API / OpenAI API

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/himanshuvyass7459/PREP-ME.git
cd prepme
```

### Install Dependencies

#### Frontend

```bash
cd client
npm install
npm run dev
```

#### Backend

```bash
cd server
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file in the server directory and add:

```env
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_api_key
```
